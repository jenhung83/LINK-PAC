# LINK-PAC
此為我第一個網站，獨立設計、製作，為一頁式靜態RWD網站。

https://jenhung83.github.io/LINK-PAC/

動機：
電茂企業原有網站並非響應式，
希望能藉由RWD優化，
增加網站在各屏幕瀏覽便利性，進而開拓新客群。

理念：
除響應式檢閱外，
亦增添產品照片、輔助圖案元素，
幫助閱讀時能清楚理解內容，
使該企業的服務項目更一目瞭然。
